# 执照

执照是一个主命令，它拥有 绑定、解绑、每日信息、执照信息、分析、申诉等子命令。

# 绑定

绑定手游城堡账号，需要三个必写参数，ID、魂晶数量、频道类型（官服，苹果官服，4399渠道，九游渠道，OPPO渠道，vivo渠道，小米渠道，华为渠道，b站渠道，抖音渠道）。

注意：如果本QQ已经绑定了账号，那么再次使用该指令会造成覆盖！

执照 绑定 <ID> <魂晶数量> <频道类型>
例子:
(test2002账号拥有999的魂晶，并且是从4399游戏盒下载的城堡)
执照 绑定 test2002 999 4399渠道

copy success

# 解绑

解绑是从机器人上解除绑定，并移除所有信息的一个指令，无需带任何参数。

注意：使用该命令需要先绑定账号！

执照 解绑

copy success

# 申诉

如果账号被别人绑定，可以使用本指令来强制夺回。 参数含有三个必填项，ID、宝藏手册（图鉴）里的怪物名称与其对应的击败数量。

执照 申诉 <ID> <怪物名称> <数量>
例子:
(test2002账号击杀哥布林盗贼50只)
执照 申诉 test2002 哥布林盗贼 50

copy success
