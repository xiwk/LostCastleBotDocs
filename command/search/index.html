# 搜索

搜索是一个搜索城堡账号执照信息的一个主命令。它拥有三个子指令：名称、ID、排行榜等。

# 名称

拥有1个必填参数：名称与1个可选参数：服务器类型。

搜索 名称 <名称> [服务器类型]
例子：
1.搜索城堡渠道账号名称为test的执照信息（未绑定）
搜索 名称 test 渠道
2.搜索城堡渠道账号名称为test的执照信息（已绑定渠道账号）
搜索 名称 test

copy success

# ID

拥有1个必填参数：ID和1个可选参数：频道类型（官服，苹果官服，4399渠道，九游渠道，OPPO渠道，vivo渠道，小米渠道，华为渠道，b站渠道，抖音渠道，默认官服）

搜索 ID <ID> [渠道类型]
例子：
1.搜索城堡4399渠道账号ID为test2002的执照信息（未绑定）
搜索 名称 test2002 4399渠道

copy success

# 排行榜

拥有1个必填参数：位次和2个可选参数：排行榜类型名称（冒险，乱斗，连战时间，连战层数，勇者试炼计时，默认为连战时间）与服务器类型（官服，渠道，默认为官服，如果你已经绑定了账号，那么会使用你当前绑定账号的服务器类型）。

搜索 排行榜 <位次> [排行榜类型] [服务器类型]
例子：
1.查询渠道连战时间排行榜榜1的执照信息（已绑定渠道账号）
搜索 排行榜 1
copy success
