# 排行榜

排行榜（榜）是一个查询游戏中榜单的一个主命令，它拥有俩个可选参数：排行榜类型名称（冒险，乱斗，连战时间，连战层数，勇者试炼计时，默认为连战时间）与服务器类型（官服，渠道，默认为官服，如果你已经绑定了账号，那么会使用你当前绑定账号的服务器类型）。

排行榜 [服务器类型] [服务器类型]
例子：
1.查询连战时间渠道榜（未绑定）
排行榜 连战时间 渠道
2.查询连战时间渠道榜（已绑定渠道账号）
排行榜
